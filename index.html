Project

Movie API

Movie Search Web App

외부 영화 API를 활용해 영화 정보를 검색하고 확인할 수 있도록 제작한 웹 애플리케이션입니다. 비동기 데이터 처리와 동적 렌더링을 경험하는 데 초점을 두었으며, 사용자가 필요한 정보에 빠르게 접근할 수 있도록 UI를 단순화했습니다.

Overview

영화 정보를 탐색하는 과정에서
사용자가 원하는 콘텐츠에 자연스럽게 몰입할 수 있도록 설계한
Movie API 기반 웹 프로젝트입니다.

TMDB API를 활용해 현재 상영작, 인기 영화, 추천 영화를 제공하며
특정 영화에 머무를 경우 관련 YouTube 트레일러 영상이 함께 노출되어
정보 탐색에서 감상으로 이어지는 흐름을 목표로 했습니다.

CONCEPT EXPLAIN

영화 선택 이후에도 사용자의 관심이 끊기지 않도록
‘머무는 시간’을 중심으로 한 인터랙션 구조를 설계했습니다.

영화 카드 선택 → 상세 정보 확인 →
일정 시간 체류 시 트레일러 영상 자동 노출이라는 흐름을 통해
능동적인 클릭 없이도 콘텐츠 소비가 이어지도록 구성했습니다.
